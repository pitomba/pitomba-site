Features

  • Synchronous

    Pitomba provides a Sprite class that can be instantiated and used to generate sprite as user (or another automatic process) demands it via an HTML page, for example.

    Demo
  • Asynchronous

    Also, pitomba provides a watcher that listens to your filesystem and generates a new sprite as soon as a image is updated. It's very similar to what compass does.
